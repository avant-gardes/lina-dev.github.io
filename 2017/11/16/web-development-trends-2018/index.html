8 Top Web Development Trends in 2018

  2017-11-16




Introduction
Web 기술은 빠르게 발전하고 있습니다. 어제 인기 있던 기술들이 내일도 인기 있다는 보장은 없습니다. Web 개발 업계는 불안정성에 대해 주목하고 있습니다. 일부 기술은 변화할 수도 있고 아닐 수도 있습니다. 이 기사에서는 2018년의 웹 개발 트랜드에 대해서 자세히 설명합니다.



JavaScript
Java Script는 2017년에 가장 유명한 웹 개발 트렌드이며 계속 발전하고 있습니다. Java Script Framework와 라이브러리는 더 유연하고 강력해질 것입니다.
Angular와 React 간의 전쟁은 뜨겁습니다. 두 프레임 워크는 장단점을 가지고 있습니다. Angular는 많은 팬이 있지만 React는 군대와 같습니다. React의 인기가 Angular 보다 우세하므로 Google Trends에 따르면 2018년에 상황이 완전 바뀔 가능성이 있다고 합니다.



Stack Overflow의 리포트에 따르면 JavaScript는 2017년에 가장 인기 있는 프로그래밍 언어입니다. 실제로는 올해가 아니라 과거 5년 동안 챔피언입니다. 이 조사에는 6만 4000명의 개발자가 참가했습니다.




SPA(Single Page Application)
JavaScript를 기반으로 한 SPA는 Web개발 트렌드에서 계속 인기를 끌고 있습니다. 모든 사용자의 디바이스에서 high-performance를 보여주고, 웹 개발의 시간을 단축해주는 것은 놀라운 것은 아닙니다. SPA의 인기는 개발자에게는 뚜렷하지만, 전체 사용률에 대해 말하기엔 아직 이릅니다. 기본적으로는 개발은 React와 Angular framework를 기본으로 합니다.



PWA(Progressive Web Apps)
Progressive Web App은 Google이 2015년도에 발표했습니다. 2016년에 일부 상위 기업이나 브랜드는 PWA에 의존하며 웹 사이트를 구축했습니다. 지난 몇 년간의 리포트에서 제시된 것처럼 Mobile App은 과포화 상태이기 때문에, PWA는 탁원한 선택지입니다.
하지만 이 application이 존속되느냐, 존패되느냐는 2018년에 나타날 것입니다.





E-commerce
E-commerce web 개발은 독자적인 룰과 법률에 의해서 구축된 독립적인 업계입니다. Website 개발 process 중 가장 보수적입니다. 하지만 적어도 Google 같은 회사에서 자사 제품을 출시하기 전까지 이 분야를 대표하는 가장 강력한 제품일 것입니다. 개발자는 거대하고 복잡한 프로젝트에는 Magento를, 작은 것에는 Shopify를 선호합니다. 2018년의 가장 뛰어난 전자 상거래 플랫폼을 분석한 기사를 볼 수 있습니다. 사이트에서는 사용자와 선호도에 대한 상세한 정보를 수집할 것입니다. 행동 이력을 기반으로 각자에게 맞는 컨텐츠를 제공할 것입니다. 온라인 스토어는 방문자에 대해 더 잘 알고 관련있는 상품만 제공할 것입니다. 이는 당신의 이름과 구입하고 싶은 것을 알고 있는 판매자의 가게로 가는 것과 같습니다.



Bots(Artificial intelligence)
인공 지능을 기반으로 한 채팅 봇이 2017년에 믿어지지 않을 정도로 어떻게 인기를 얻었는지 알 것입니다. 페이스 북의 봇이 어떻게 서로 의사 소통을하면서 자신의 언어를 만들었는지에 대한 이야기를 기억하십시오. 이런 놀라운 이야기는 우리가 새로운 변화 위기에 직면하고 있음을 나타내고 있습니다. 인공 지능(artificial intelligence)과 신경 회로망(neural networks)에 근거하여 만들어진 채팅봇은 온라인 커뮤니케이션을 증가시키기 위해 계속 발전할 것입니다. 이것이 2018 웹 개발 트렌드와 이어질 것이라고 생각합니다.



Blockchain
Blockchain이 웹 산업 전반에 미치는 영향에 대해 많은 사람들이 알고 있다고 생각합니다. 간단하게 설명하고자 합니다.


블록 체인은 공통 데이터 스토리지의 방법입니다. 모든 정보는 전 세계의 수 천개의 컴퓨터 상에 놓여 있고 한 장소에 있진 않습니다. 거래 사이에 중개자가 존재하지 않는 것이 주된 이점입니다. 전 세계의 수백 대의 컴퓨터에 의지하는 복잡한 알고리즘으로 모든 거래를 검증합니다. Blockchain은 높은 레벨의 보완 가지고 있습니다. 따라서 세계 각국의 은행에서 Blockchain을 도입 가능성을 검토하고 있습니다.





Internet of Thing
IoT솔루션은 점차 우리의 삶 속에 스며들고 있습니다. 우리는 스마트 기기의 세계적인 도입을 앞두고 있습니다. 더욱이, 수년 전에 장치가 엔터테인먼트에 사용 되었다면, 이제 IoT는 경제의 다른 분야로 대규모 국제 기업에서 진행하기 시작했습니다. 스마트 기기는 좀 더 효율적으로 리소스를 사용하도록 도와줍니다. IoT는 무한한 비지니스 기회를 열고, 높은 정확성과 효율성을 증가시킵니다. Statista에 따르면 2025년에는 기기의 수가 3.5배로 늘어난다고 합니다.





Motion UI
웹 사이트에서 가장 매력적인 staff은 부드러운 애니메이션입니다. 틀림없이 2018년의 웹 디자인의 트렌드 중 하나일 것입니다. 사용자가 사용하기 좋은 작업이 GIF 애니메이션이이냐 Motion UI는 중요하지는 않습니다. 하지만 Motion UI는 개발자가 JavaScript나 jQuery 라이브러리에 대한 많은 지식을 가지지 않아도 쉽게 콘텐츠를 애니메이션화할 수 있어 2017년에 가장 인기 있는 라이브러리입니다. 개발은 SASS preprocessor에 의존하고 있습니다.




이것을 마지막으로 2018년 웹 개발 트랜드를 마칩니다. 미흡한 번역이나마 도움이 되었으면 좋겠습니다!
